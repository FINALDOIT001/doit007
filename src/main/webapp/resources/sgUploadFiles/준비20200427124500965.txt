안녕하세요. IT 자유 주제를 공부해서 발표하는 스터디를 모집하고 있습니다.

(굳이 IT 자유 주제가 아니여도 됩니다. 영어, 자격증 공부, 기타 등등 가능합니다.)

스터디의 목적은 각자 스스로 공부하는 것이 목적입니다.

발표를 안해도 되며, 편한 마음으로 들으러 오셔도 됩니다.



제가 발표할 내용은 LeetCode 문제를 풀고 발표할 예정입니다.







1. 스터디 내용 : IT 관련 자유 주제를 공부해와서 발표하는 것

2. 시간 : 매주 토요일 오전 10~12

3. 장소 : 신촌역 토즈 스터디

4. 비용 : 스터디 사용료 ( 1/n )

- 한 달치 미리 걷고 시작합니다.(예약했는데 도중에 안나오시는분들이 계셔서..)

5. 연락처 : rainbowblue@naver.com  

같이 스터디 하실 분은 

연락처 / 성함 / 메일주소 / 발표할 주제 / 지금 하고 계시는 일 

적어서 메일로 보내주시면 감사하겠습니다.

/**
    * 디테일 상세내용 / 참석자
    * 작성자 : 서정도
    * @param mv
    * @param sgNo
    * @param currentPage
    * @return
    */
   @RequestMapping("studyDetail.go")
   public ModelAndView studyDetail(ModelAndView mv, int sgNo,
         @RequestParam(value = "currentPage", required = false, defaultValue = "1") int currentPage) {

      StudyGroup sg = sgService.selectSg(sgNo);
      
      ArrayList<Member> ml = sgService.memberList(sgNo);
      ArrayList<Gallery> galleryList = sgService.GalleryList(sgNo);
      
      System.out.println("Controller memberList : " + galleryList);
      
      System.out.println("Controller memberList : " + sg);
      System.out.println("Controller memberList : " + ml);
      
      if (sg != null) {
         mv.addObject("sg", sg)
           .addObject("ml",ml)
           .addObject("currentPage", currentPage)
           .addObject("galleryList", galleryList)
           .setViewName("study/doitStudyDetail");
      } else {
         mv.addObject("msg", "게시글 상세조회 실패").setViewName("common/errorPage");
      }
 
      return mv;
   }
